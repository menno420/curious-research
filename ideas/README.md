# Ideas — one file per idea, any size, zero obligation

Drop one-liners here (ask Claude: *"add an idea: …"*). Grow one whenever you feel like it
with the ritual in [`docs/idea-ritual.md`](../docs/idea-ritual.md) — it ends in
`build / park / drop / think more`, and every verdict including *drop* is a win.

**Seeded starters** (matched to your gear — a menu, not a to-do list):
