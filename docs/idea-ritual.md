# The idea ritual — 8 questions, one page, one verdict

> **Status:** `reference`

Drop a one-liner in [`ideas/`](../ideas/) whenever something crosses your mind. When you feel
like growing one, tell Claude: *"run the ritual on `ideas/<file>`."* It fills the idea file
with the 8 answers and ends in exactly one verdict. Nothing here is homework — un-probed
ideas are fine forever.

1. **What is this thing, really?** (one honest sentence)
2. **What could it grow into?** (the fun ceiling)
3. **What's the coolest version of the simplest build?**
4. **What breaks it?** (printability, tolerance, torque, time, physics)
5. **What does it let me build next?** (does it compound?)
6. **What does it need?** (parts, filament, libraries, skills)
7. **Who does what — me, Claude, or both?**
8. **What's the smallest piece I could finish this weekend?**

**Verdict (pick one):** `build` (Claude sets up the project + first steps) · `park` (good,
not now) · `drop` (honestly no — also a win) · `think more` (name the one open question).

House honesty rule: a true "this isn't worth it" beats an enthusiastic maybe. **No forced
projects — an empty week is fine.** Write "printed nothing, tuned nothing" rather than
inventing activity.
