What the animation simplifies: real arms mount the plate at whatever angle the wrist servo is turned to — this side view flattens that to keep the idea clear. The pen tip and magnet are drawn big so you can see them. And two tight screws already resist turning; the hub recess and back-notch shown here are what make the fit centred and repeatable, so every tool lands in the same place facing the same way.

What you just watched

  • One plate bolts to the arm's servo horn: two screw holes (the bolt pattern), a round hub recess, and a back-edge notch.
  • Any tool — the pen holder, the magnet pickup — is built on that same plate, so it mounts in the same two holes, in the same spot.
  • The key (hub recess + notch + two screws) stops the tool spinning or sitting crooked, so it always faces the same way.
  • The payoff: design a new tool once and it just clicks on. Swapping is two screws, not a re-measure.

Built as source only — the shapes live in mount_standard.scad and magnet_tool.scad next to this file. You render, slice, and print them yourself; any lift is a "check the magnet holds your part first."
